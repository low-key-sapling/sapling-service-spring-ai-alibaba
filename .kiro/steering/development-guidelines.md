# 开发规范指南

## 技术栈

- Java 版本: 1.8
- Spring Boot 版本: 2.7.18
- 构建工具: Maven

## 代码规范

1. 使用 Lombok 简化代码
2. 使用 MapStruct 进行对象映射
3. 所有文件使用 UTF-8 编码

## 安全规范

1. 最小化依赖原则
2. 密码加密存储
3. 关闭非必要的 Actuator 端点
4. API 接口加密
5. 日志级别控制

## 监控规范

1. 队列监控（内存队列/Kafka LAG）
2. 使用 KoTime 进行性能追踪（默认关闭）
3. 日志滚动和级别控制

## 部署规范

1. 多环境配置管理
2. 使用 Nginx 进行反向代理
3. 自定义构建流程
4. UOM + 制品包管理
